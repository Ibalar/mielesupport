# Исправление горизонтального скролла в блоке Reviews

## Внесенные изменения:

### 1. `/wp-content/themes/miele-support/assets/css/reviews.css`

**Изменения в `.reviews__wrapper`:**
- Удалён `cursor: grab` из базовых стилов (мешал нативному скроллу)
- Добавлен `scroll-behavior: smooth` для плавной прокрутки
- Добавлен `user-select: none` для предотвращения выделения текста при драге
- Изменён `.reviews__wrapper:active` на `.reviews__wrapper.active` (для работы с JS)

**Добавлены стили для десктопа (1024px+):**
- `.reviews__wrapper:hover { cursor: grab }` - показываем grab только при наведении
- `.reviews__wrapper:active, .reviews__wrapper.active { cursor: grabbing }` - курсор при драге

### 2. `/wp-content/themes/miele-support/assets/js/reviews.js` (новый файл)


**Функционал drag-скролла:**
- Отслеживание нажатия мыши (mousedown)
- Плавное перетаскивание (mousemove с множителем скорости x2)
- Добавление/удаление класса `active` для изменения курсора
- Очистка состояния при отпускании мыши или выходе курсора

**Функционал wheel-скролла:**
- Перехват события `wheel`
- Конвертация вертикального скролла в горизонтальный
- Работает только если вертикальная прокрутка больше горизонтальной
- Использует `passive: false` для возможности preventDefault

### 3. `/wp-content/themes/miele-support/functions.php`

```php
theme_script('reviews-js', 'reviews.js');
```

## Результат:

 **Горизонтальный скролл колесиком мыши** - работает на всех устройствах
 **Drag (перетаскивание) мышкой** - работает плавно с визуальной обратной связью
 **Smooth scroll** - плавная анимация прокрутки
 **Адаптивность** - корректная работа на мобильных и десктопных устройствах
 **UX улучшения**:
  - Курсор `grab` появляется только при наведении на десктопе
  - Курсор меняется на `grabbing` при активном драге
  - Текст не выделяется при перетаскивании
  - Скорость драга настроена (множитель x2)

## Техническая информация:

- Используется нативный `overflow-x: auto` для базовой функциональности
- JavaScript добавляет расширенные возможности (drag и wheel)
- Код совместим со всеми современными браузерами
- Использованы оптимальные настройки `passive` для производительности
